# 作业计划表 · 使用说明

面向家长/学生的日常使用指南。安装与部署请参阅 [安装说明.md](./安装说明.md)。

---

## 1. 登录与账号

| 入口 | 地址（本机） |
|------|----------------|
| 用户端 | http://localhost:5173 |
| 登录/注册 | http://localhost:5173/login |
| 运营后台（PC） | http://localhost:5173/admin |

- 首次使用可 **注册** 普通账号，登录后只能管理 **自己的计划**。
- 管理员账号由系统在首次启动时创建（默认见 `.env` 中 `ADMIN_USERNAME` / `ADMIN_PASSWORD`），可在运营后台管理全站用户与计划。
- 同一局域网内其他设备访问：启动后 `start.bat` 会显示 `http://本机IP:5173`。

---

## 2. 计划（学期/阶段）

1. 底部进入 **「我的」** → **计划管理**。
2. **新建计划**：填写名称、说明、截止日期等。
3. 可从 **课表模板** 一键生成（含示例地点与周课表）。
4. 顶部 **计划切换器** 可在多个计划之间切换。
5. 支持复制计划、归档/删除计划。

每个计划的数据（课表、任务、附件等）相互独立。

---

## 3. 底部导航

| 标签 | 功能 |
|------|------|
| **总览** | 今日安排、各科进度、距截止天数等仪表盘 |
| **每日** | 选定日期的课表 + 长期作业进度记录 |
| **课表** | 周课表维护（列表/表格两种视图） |
| **任务** | 各科长期作业（书名/作业名、总量、附件） |
| **我的** | 计划、课程、地点、作业安排、账号设置 |

---

## 4. 周课表

- **添加时段**：选择星期（可多选）、时间、类型、课程或作业安排、地点。
- **类型**
  - **培训班**：对应「课程管理」中的培训班，一般只需出现在课表上。
  - **自安排**：需在「每日」里写当天具体做什么。
- **表格视图**：横向为星期、纵向为时间；点击单元格可编辑。
- **列表视图**：按星期分组；**左滑** 或 **长按** 可编辑/删除。
- **每日临时调整**：在「每日 → 当日课表」中，对培训班卡片左滑/三点菜单可改时间、取消、恢复等。

---

## 5. 长期任务与附件

在 **「任务」** 中按 **作业安排**（科目/类别）分组管理：

- 新建任务：书名/作业名、总数量与单位（如「14 节」）、说明。
- **附件**：支持 PDF、Word、Excel、PPT、图片；可多选一次上传多个；上传有进度条。
- 附件可 **重命名**、**预览**（缩放/滚动/选中复制文字）、**移除**。
- 列表项 **左滑** 编辑或删除。

附件大小上限默认 **100MB**，管理员可在 `/admin/settings` 修改。

---

## 6. 每日计划

1. 顶部切换日期（左右箭头或点日期选择）。
2. **当日课表**：显示该日培训班与自安排；自安排需填写详细计划。
3. **长期作业进度**（可选）：为某任务记录当天计划完成量；**同一任务同一天只能有一条**，重复添加会提示「请直接编辑」。
4. 记录可标记 **完成** / **部分完成** / 编辑 / 删除。

---

## 7. 「我的」常用设置

| 菜单 | 说明 |
|------|------|
| 计划管理 | 新建、编辑、复制、删除计划 |
| 课程管理 | 培训班名称、老师、默认地点等 |
| 上课地点 | 地点名称与地址 |
| 作业安排 | 各科/类别（数学、语文等），任务会按此分组 |
| 账号设置 | 修改昵称、密码、计划截止日期等 |

---

## 8. 运营后台（管理员）

在 PC 浏览器打开 `/admin`：

- **概览**：用户数、计划数等统计
- **用户**：启用/禁用账号、查看用户计划
- **计划**：查看全站计划详情
- **设置**：附件上传大小限制等

---

## 9. 常见问题

**Q：手机打不开页面？**  
A：电脑与手机需在同一 WiFi；防火墙需放行 5173 端口；使用 `start.bat` 显示的局域网地址。

**Q：上传附件失败？**  
A：检查文件格式与大小；确认后端（8002 端口）已启动；重启 `start.bat` 后再试。

**Q：课表单元格出现两条相同内容？**  
A：多为重复保存导致；左滑删除多余项。新版本已防重复提交并自动清理历史重复数据。

**Q：忘记管理员密码？**  
A：在服务器上修改 `.env` 中 `ADMIN_PASSWORD`，删除数据库中 admin 用户的 `password_hash` 后重启，或联系维护人员重置。

---

## 10. 数据说明

- 用户、计划、课表、任务等存在 **数据库**（本地默认 SQLite 文件 `backend/homework.db`，生产可用 PostgreSQL）。
- 任务附件保存在 `backend/uploads/{计划ID}/`，随计划删除需另行清理文件（如有需要）。

更多安装、迁移、服务器部署步骤见 **[安装说明.md](./安装说明.md)**。
